# temperaturna-plosca
Project in a Distributed systems course.

## Compiling
\[Linux\]```gcc temp_plosca.c -o temp_plosca \`pkg-config --cflags gtk+-3.0\` \`pkg-config --libs gtk+-3.0\` -lm```

## Libs
[Cairo](https://www.cairographics.org/) for visualization.
